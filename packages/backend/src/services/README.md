# Services

This directory contains business logic services that interact with the Oracle database packages.

Services handle:

- Database operations via Oracle PL/SQL packages
- Business logic implementation
- Data transformation
- Error handling and validation

Service files:

- `AccountService.ts` - Calls p_accounts package
- `FeatureService.ts` - Calls p_features package
- `SessionService.ts` - Calls p_sessions package
- `AuthService.ts` - Authentication logic
